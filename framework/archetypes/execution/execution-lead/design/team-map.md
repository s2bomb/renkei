# Team Map: Execution-Lead

**Date**: 2026-02-26

## Role Type

`execution-lead` is a **team leader** and **stage owner** for the execution stage.

## Parent and Children

- Parent leader: `tech-lead`
- Child members:
  - `test-implementer`
  - `implement-plan`
  - `validate-plan`

## Stage Boundary

- Stage entry: handoff package acknowledged from `tech-lead`.
- Stage exit: validation confirms implementation meets plan and source requirements.

## Handoff Direction Rules

1. Member outputs return to `execution-lead`.
2. `execution-lead` synthesizes execution status and validation outcome.
3. Only `execution-lead` publishes stage-complete report.

## Satellite Usage

Satellite teams may be delegated for focused diagnostics or research. Outputs return to `execution-lead` as requesting parent leader.
